Projects

A deep dive into a few of my favorite personal projects

AVL Tree

A self-balancing binary search tree

  • Self-balances through 90-degree rotations, which are performed as necessary as each new node is added
  • Recursive search functionality with time complexity of O(log n)

Minesweeper

A recreation of the classic computer game

  • Fully functional, made from scratch using C++ and SFML
  • Utilizes object-oriented programming to track relevant information
  • Recursively clears adjacent tiles as appropriate, staying true to one of the game's core features

PageRank

A simplified webpage ranking system inspired by Larry Page's

  • Determines ranks from a directed, adjacency-list based graph of webpages
  • Derives a value for each page based on how many other pages reference it, and the value of those pages
  • Scalable: capable of functioning correctly with graphs of large sizes (since the internet is a pretty big place)
